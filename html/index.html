App Studio Demo Application

Use App Studio to Build your Application

This application is a standard Node.js HTTP application packaged into a single container. And has now been edited by Will Wilson

Demo-palooza! Run this and more using a script launcher found here https://github.com/jduimovich/appstudio-e2e-demos

Try

  1. Bootstrap your cluster from https://github.com/redhat-appstudio/infra-deployments 
     ./hack/bootstrap.sh
  2. Create an Application and a Component 
    3.     apiVersion: appstudio.redhat.com/v1alpha1
    kind: Application
    metadata:
      name: single-nodejs-app
    spec:
      description: "single-nodejs-app built in App Studio"
      displayName: single-nodejs-app 
    
        apiVersion: appstudio.redhat.com/v1alpha1
    kind: Component
    metadata:
      name: single-nodejs-app
    spec:
      componentName: single-nodejs-app
      application: single-nodejs-app
      containerImage: quay.io/jduimovich0/single-nodejs-app
      source:
        git:
          url: https://github.com/jduimovich/single-nodejs-app
          devfileUrl: https://raw.githubusercontent.com/jduimovich/appstudio-e2e-demos/main/demos/single-nodejs-app/devfiles/devfile.yaml
      build:
        containerImage: quay.io/jduimovich0/single-nodejs-app
  3. Navigate to the topology viewer and your app will be running in your namespace.
